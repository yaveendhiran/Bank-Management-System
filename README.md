# Bank Management System in C

## Project Description
A console-based Bank Management System developed in C using structures and file handling.

## Features
- Insert new account
- Display all account records
- Deposit money
- Withdraw money
- Search account by account number
- Delete account record

## Technologies Used
- C Programming
- File Handling
- Structures

## How to Run
Compile:
gcc bank.c -o bank

Run:
.\bank.exe
